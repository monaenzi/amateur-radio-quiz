# amateur-radio-quiz

A web app for preparing for the Austrian amateur radio exam (ÖVSV).

## About

This app helps students prepare for the Austrian amateur radio license exam. 

> This app is tailored to Austrian exam content and regulations, which differ from other countries.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
