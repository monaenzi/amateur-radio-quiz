# Contributing

## Branching
- `main` — stable, protected
- Branch naming: `feature/short-description`, `fix/short-description`, `chore/short-description`

## Commits
Follow [Conventional Commits](https://www.conventionalcommits.org/):
- `feat:` new feature
- `fix:` bug fix
- `chore:` maintenance/setup
- `docs:` documentation only

## Pull Requests
- Every PR must reference an issue (`Closes #123`)
- At least 1 review required before merging into `main`

## Issues
- Use the provided templates for User Stories and Technical Stories
